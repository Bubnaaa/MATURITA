## Charakteristika, architektura, historie, Windows/Linux/…, multitasking, firmware, instalace, konfigurace
---
### Operační systém
- Základní SW řídící počítač
- Zprostředkovává komunikaci mezi HW a aplikacemi
- Řídí procesor, paměť, disky, síť, vstupní/výstupní zařízení
- Poskytuje CLI/GUI
### Funkce OS
- Správa procesů - spouštění, ukončování, přepínání kontextu
- Správa paměti - přidělování RAM, paging, ochrana paměti
- Správa souborů - souborové systémy (NTFS, FAT32)
- Správa vstupně-výstupních zařízení - ovladače, komunikace s HW
- Bezpečnost - uživatelské účty, oprávnění, šifrování
- Síťové služby - TCP/IP, DNS, DHCP klient,...

## Architektura OS
- Jádro (kernel) - základ OS, správa procesů, pamětí
	- Typy kernelu:
		- Monolitické (Linux)
		- Mikrojádro (Minix)
		- Hybridní (Win NT)
- Systémové knihovny
- Ovladače zařízení 
- GUI

---
### Historie OS
- 50.-60. léta - Dávkové systémy na sálových počítačích
- 70. léta - vznik UNIXu - základ pro dnešní OS
- 80. léta - MS-DOS (textový), první GUI systémy (MacOS)
- 90. léta - Windows 95 - GUI + první multitasking
- 2000+ - Linuxové distribuce, Windows NT, mobilní OS (Android, iOS)

---

### Multitasking
- = schopnost OS spouštět a pracovat na více úlohách najednou
- Preemptivní multitasking- OS vybírá procesy podle plánovače (Windows, Linux)
- Kooperativní multitasking - procesy uvolňují CPU samy (staré systémy, nestabilní)
- Plánování procesů - FIFO, Round Robin, prioritní plánovače

### Nejznámější OS

#### Windows
- Uzavřený komerční OS od Microsoftu
- Pro běžné uživatele, školy, firmy
- Výhody:
	- Široká podpora HW i SW
	- Snadné ovládání
- Nevýhody:
	- Vyšší nároky na HW 
	- Menší kontrola nad systémem
	- Častější cíle útoků
	- Cena

#### Linux
- Otevřený systém s open source jádrem 
- Distribuce: Ubuntu, Debian, Fedora, Mint, Arch, Kali,...
- Výhody:
	- Stabilita
	- Bezpečnost
	- Upravitelnost
	- Zdarma
- Nevýhody:
	- Horší podpora některých aplikací
	- Potřeba znalostí

#### Další OS

### Firmware
- Nejnižsí SW uložený v paměti ROM/Flash
- Řídí základní funkce zařízení při startu
- Např.: BIOS, UEFI, firmware routerů, firmware SSD,...
- Vlastnosti: Běží před OS, umožňuje inicializaci HW, někdy lze aktualizovat

### BIOS & UEFI
- BIOS - starší systém, omezený, pracuje v 16-bit režimu
- UEFI - moderní nástupce, rychlejší start, podpora velkých disků, Secure boot, grafické menu
